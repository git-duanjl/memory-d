	3. 配置mysql
		1. 使用MySQL Yum库 -- https://dev.mysql.com/doc/mysql-yum-repo-quick-guide/en/
			1.1 mysql yum资源库下载地址 ： http://dev.mysql.com/downloads/repo/yum/
			1.2 sudo rpm -Uvh platform-and-version-specific-package-name.rpm -- Yum安装
			1.3 yum repolist all | grep mysql -- 查看mysql 资源库 服务状态
			1.4 启动禁用服务
				sudo yum-config-manager --disable mysql57-community
				sudo yum-config-manager --enable mysql56-community
			1.5 /etc/yum.repos.d/mysql-community.repo -- 发布子库
			1.6 sudo yum install mysql-community-server -- install mysql
			1.7 start
					sudo service mysqld start
					sudo systemctl start mysqld.
			1.8 status
					sudo service mysqld status
					sudo systemctl status mysqld.service
			1.9 password log
					sudo grep 'temporary password' /var/log/mysqld.log
			2.0 login 
					mysql -uroot -p
			2.1 set password
					ALTER USER 'root'@'localhost' IDENTIFIED BY 'MyNewPass4!';
			2.2 mysql_secure_installation
					允许您执行重要的操作，如设置root密码，删除匿名用户等等。总是运行它来保护你的MySQL 5.6安装
			2.3 yum --disablerepo=\* --enablerepo='mysql*-community*' list available
					列出可用于您的平台的所有MySQL组件的包
			2.4 sudo yum install package-name
					安装 package-name 软件包
			2.5 sudo yum update mysql-server -- 升级mysql及其组建
			2.6 sudo yum update mysql-server -- 列出MySQL组件的所有已安装软件包
			2.7 sudo yum update package-name -- 更新软件包
		2.1 数据备份
			 FLUSH TABLES tbl_list WITH READ LOCK;  -- 通过复制表文件进行备份 ※ 通过复制表文件进行备份
			 SELECT * INTO OUTFILE 'file_name' FROM tbl_name --  此方法适用于任何类型的数据文件，但仅保存表数据，而不保存表结构
